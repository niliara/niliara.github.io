Variables en C#

C# es un lenguaje tipado, por lo que hay que especificar el tipo de variable antes de declararla.

Por ejemplo, vamos a crear una variable integer de valor 5:

int numeroCinco = 5;

Tipos de variables

Hay muchos tipos de variables en C#, pero los principales son:

  • int : Integer, numero entero
    • int nom = 6;
  • float : Float, numero decimal, se separa con un punto
    • float nom = 6.2;
  • bool : Boolean, puede ser true o false
    • bool nom = true;
bool nom = false;
  • string : Variable de texto, se declara con el uso de comillas
    • string nom = "no hi ha pressupost per wifi";

Pedir valores al usuario

Para pedir un valor al usuario, usaremos Console.ReadLine, que devuelve lo que el usuario escriba en forma de string.

Console.WriteLine("Dame un numero");
string answer = Console.ReadLine();

En este ejemplo, el programa pide al usuario una respuesta y la almacena en la variable answer.

Si queremos ver el valor, se puede imprimir de la siguiente manera:

Console.WriteLine(answer);

Convertir string a int

Podemos convertir el string del ejemplo anterior a una variable de tipo integer utilizando int.Parse:

int converted = int.Parse(answer);

Convert.ToInt32 también hace la misma función:

int converted = Convert.ToInt32(answer);

Pedir un número al usuario:

int answer = int.Parse(Console.ReadLine());
