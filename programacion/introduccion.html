Introducción a C#

C# es el lenguaje que utilitzamos para aprender los fonamentos de la programación. Es un lenguaje de alto nivel y está orientado a objetos

Básicos de dotnet

Después de haber instalado dotnet, ve a un directorio vacío y ejecuta:

dotnet new console

Eso creará un nuevo proyecto. Por ahora, el único archivo que nos interesa es Program.cs.

Para ejecutar un proyecto, hay que correr el comando:

dotnet run

Para compilar un .exe, hay que utilizar el siguiente comando:

dotnet publish -r win64

o para los usuarios de windows:

dotnet publish

Hello world

Así es como se vería el programa "Hello World" en C#:

class Program
{
	public static void Main(string[] args)
	{
		Console.WriteLine("Hello world");
	}
}

Para empezar, en Program.cs tenemos que cambiarlo por la plantilla que aprendimos en clase. añadimos un ConsoleWriteLine, que es el equivalente a un "print" en python.

Después de cada comando hay que poner ; y los {} sirven para declarar qué está dentro de cada función.

Comentarios

Para poner un comentario, escribir // para marcar una línia.

//esto es un comentario

También se puede escribir /* y */ para marcar principio y final del comentario

/*esto
es
un
comentario
*/
