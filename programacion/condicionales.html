Condicionales en C#

Los condicionales manejan la toma de decisiones del programa. Dependiendo de si las condaciones se cumplen (true) o no (false), se ejecutarán unos comandos o otros.

Condicional if

bool printHelloWorld = true;

if (printHelloWorld)
{
	Console.WriteLine("Hello World!");
}

En éste ejemplo, se comprueba si la condición printHelloWorld se cumple (true) o no (false).

En éste caso, sí que se cumple (es true), por lo tanto el programa ejecutará el código dentro del if (delimitado por {}).

Condicional else

if (printHelloWorld)
{
	Console.WriteLine("Hello World");
}

else
{
	Console.WriteLine("Bye World");
}

Éste ejemplo es igual al anterior, excepto que en caso de que la condición no se cumpla, se ejecutará el código dentro de else {}.

Comparaciones

Se pueden usar comparaciones como condiciones, siendo el resultado de la comparación un valor boolean:

  • == : igual a
    • if (0 == 0) //true
  • != : no igual a
    • if (0 == 0) //false
  • < : más pequeño que
    • if (1 < 2) //true
if (2 < 2) //false
  • > : más grande que
    • if (2 > 1) //true
if (2 > 2) //false
  • < : más pequeño o igual que
    • if (1 <= 2) //true
if (2 <= 2) //true
  • > : más grande o igual que
    • if (2 >= 1) //true
if (2 >= 2) //true

Ejemplo: comprobar si el usuario es mayor de edad

Console.Write("Dime tu edad: ");
age = int.Parse(Console.ReadLine());

if (age >= 18)
{
	Console.WriteLine("Eres mayor de edad");
}

else
{
	Console.WriteLine("Eres menor de edad");
}
