Introduccion a C#

C# es el lenguaje que utilitzamos para aprender los fonamentos de la programación. Es un lenguaje de alto nivel y está orientado a objetos

Básicos de dotnet

Después de haber instalado dotnet, ve a un directorio vacío y ejecuta:

dotnet new console

Eso creará un nuevo proyecto. Por ahora, el único archivo que nos interesa es Program.cs.

Para ejecutar un proyecto, hay que correr el comando:

dotnet run

Para compilar un .exe, hay que utilizar el siguiente comando:

dotnet publish -r win64

o para usuarios de windows:

dotnet publish
