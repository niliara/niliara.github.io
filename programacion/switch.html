Switch en C#

Los switch son condicionales que se usan para manejar muchas posibilidades a la vez. De esta forma, podemos evitar cadenas de if / else.

Por ejemplo, este código:

if (color == "red")
{
	Sprite.modulate = "#cc0000";
}

else
{
	if (color == "green")
	{
		Sprite.modulate = "#00cc00";
	}
	
	else
	{
		if (color == "blue")
		{
			Sprite.modulate = "#0000cc";
		}
	}
}

...puede substituirse por: 

switch (color)
{
	case "red":
		Sprite.modulate = "#cc0000";
		return;
	case "green":
		Sprite.modulate = "#00cc00";
		return;
	case "blue":
		Sprite.modulate = "#0000cc";
		return;
}

...donde "color" es la variable en la que se aplica y "case" es para administrar los valores posibles y el codigo que ejecutan.

Es obligatorio poner "return" después de cada caso.

En vez de case, podemos usar "default", que funciona como un "else" (se ejecuta si no se cumple ninguna).

default:
	Sprite.modulate = "#000000";
	return;
