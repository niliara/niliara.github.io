Bucles en C#

Los bucles sirven para repetir fragmentos de código sin necesidad de reescribirlos varias veces.

Bucles while

Mientras la condición se cumpla, se va a seguir ejecutando el código dentro del while {}

while (true)
{

}

En este ejemplo, hay un bucle infinito (hasta que algo lo rompa).

Bucles for

Los bucles for sirven para repetir un fragmento de código una cantidad de veces especificada. Funciona de forma similar al while:

for (int i = 1; i < 10; i++)
{
	Console.WriteLine(i)
}
  • Primero crea una variable integer con el nombre i
    • ... int i = 1 ...
  • luego, se indica la condición para que se repita el código
    • ... i < 10 ...
  • por úlimo, se declara que después de cada repetición le sumamos 1 a la variable i
    • ... i++ ...

En el ejemplo anterior, la i empezaría en 1 y sumaria 1 después de cada repetición. Al llegar a 10, i deja de cumplir la condición i < 10, y el bucle termina.

Output:
1
2
3
4
5
6
7
8
9

Romper un bucle

Para romper un bucle for o while, se puede usar el comando break

break;
